# 🟡 Pokédex – Version 1.0

![Pokédex Logo](./assets/img/pokédex.png)

## 📖 Über das Projekt

**Pokédex** ist mein persönliches Frontend-Projekt, das auf der **PokéAPI** basiert. Ziel des Projekts ist es, eine moderne und benutzerfreundliche Webanwendung zu entwickeln, mit der Pokémon durchsucht und deren Informationen übersichtlich dargestellt werden können.

Version **1.0** bildet das Grundgerüst der Anwendung und beinhaltet bereits die komplette Seitenstruktur mit Header, Footer sowie einem Suchbereich. In den kommenden Versionen wird das Projekt kontinuierlich erweitert.

Dieses Projekt dient gleichzeitig als praktische Übung im Umgang mit APIs, JavaScript, DOM-Manipulation und responsivem Webdesign.
